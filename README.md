# errors-as-values

### dev

To install dependencies:

```bash
bun install
```

### test

```bash
bun run build
```

This project was created using `bun init` in bun v1.0.2. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

# Deployment

- Build with `bun run build`
- Go into `dist`
- Run `npm version <patch|minor|major>`
- Run `npm publish`
